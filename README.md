# Weather Forecast & Traffic Cam

## Installation
`pnpm` is required for this.

```shell
# For Node >16
corepack enable
corepack prepare pnpm@<version> --activate

# For Node <16
npm install -g pnpm
```

```shell
# Install the packages
pnpm install
# Start both the server and client
pnpm start
```
